## 🌍 The Trillion-Dollar Bottleneck
Global supply chains are fundamentally dependent on maritime choke points. When the Panama Canal experiences severe droughts induced by El Niño, the Gatun Lake water levels plummet. The resulting draft restrictions and auction-slot bidding wars force vessels to either pay millions in expedite fees, wait weeks at anchorage, or undertake massive tactical detours via the Strait of Magellan. 

**NSR-Aegis** is an autonomous decision intelligence platform. By bridging macroeconomic variables, geopolitical risk, and precise hydrological physics, the Aegis Multi-Agent system decides the exact logistical route a multi-million dollar vessel should take.

## 🧠 The Agentic Neural Architecture
At the core of NSR-Aegis is the `PanamaPredictiveEngine`, an advanced Random Forest Regressor trained on highly coherent meteorological and oceanographic telemetry.

* **Hydrological Physics:** The model respects the strict physical geometry of the canal. Maximum allowable draft predictions accurately cap at **15.24 meters (50 feet)**—the official Neopanamax limit.
* **Climatic Coherence:** The engine processes tropical monsoonal precipitations (47mm in dry season up to 230mm+ in wet season) to forecast Gatun Lake operational levels precisely between the 25.9m and 26.7m critical thresholds.
* **Oceanographic Fidelity:** Visibility degradation due to heavy rainfall and ocean salinity dilution (dropping to ~34.1 PSU due to freshwater runoff) are intrinsically calculated into the transit risk index.

## ⚙️ Hardware Integration (Aegis Ice-Node)
True Naval Engineering requires a cyber-physical connection. Included in this repository (`/hardware`) is the complete **KiCad schematic** for the Aegis Telemetry Node. 
Powered by an **ESP32-WROOM-32**, an MPU-6050 accelerometer, and a NEO-M8N GNSS module, this physical node is designed to be mounted on the hull to stream live structural bending stress telemetry during tight lock maneuvers.

*Note (Rule 9 Compliance): Due to Round 1 geographical and time constraints, the live hull stress data shown in the dashboard is generated by a Stochastic Simulation Module, strictly modeled after the physical parameters of our engineered KiCad PCB design.*

## 💻 Execution & Deployment (Builder-First)
The Digital Twin interface is built for C-Level executives and Bridge Commanders.

### Setup Instructions
1. Clone the repository and navigate to the root folder.
2. Install the exact dependencies required by the Aegis environment:

```bash
pip install -r requirements.txt
```

3. Initialize the Digital Bridge / Command Center:

```bash
python panama_web_server.py
```

4. Access the Cyber-Physical Dashboard by opening `index.html` in your browser.

## 🛡️ Fallback Resiliency
In the event of API backend failure or GNSS Spoofing attacks, the Aegis interface contains mathematical offline fallbacks, including an Astronomic Navigation Module (Digital Sextant) utilizing ephemeris data for Polaris to calculate latitude independently of satellite constellations.
